# Instructions for building campaign

Services required:
1. Github.com (Free Account)
2. Cursor.com (Free Account)
3. Netlify.com (Free Account)
4. Google Ads Account (Free to Launch)

Steps:
1. Create / Sign-in to Github.com account
    - Go to project repo. (Https://github.com/ChrisChasm/ad-campaign-example)
    - Click “Use this Template"
    - Give it a new name and save to your own repo
    - Download repo to your desktop

2. Create / Sign-in to Netlify.com
	- Create a sign-in
    - Create a project
    - Choose a Github repo
    - Add ./website to the root

3. Download and install Cursor CLI (and Cursor IDE if desired)
	- Open Terminal (Mac) / Command (Windows)
    - Install Cursor CLI
    - Right click your desktop folder, open with Terminal
    - cursor-agent, accept
    
4. Open the folder with the Cursor Agent
	- Right click the folder on your desktop and select "open in terminal"
    - Type ‘cursor-agent'
    - Give permission

6. Run the following prompt step by step. Prompt.md
